# 最长公共子序列（Longest Common Subsequence, LCS）

## 一、问题定义

给定两个字符串 `text1` 和 `text2`，返回这两个字符串的 **最长公共子序列** 的长度。如果不存在公共子序列，返回 `0`。

> **子序列**：由原字符串删除某些字符（也可以不删除）而不改变剩余字符的相对顺序得到的新字符串。
> 例如，`"ace"` 是 `"abcde"` 的子序列（删除了 `'b'` 和 `'d'`）。

---

## 二、核心思想：动态规划（Dynamic Programming）

### 2.1 为什么用动态规划？

该问题具有 **最优子结构** 和 **重叠子问题** 两个特性：

- **最优子结构**：两个字符串的 LCS 可以由其前缀的 LCS 推导出来。
- **重叠子问题**：在递归地比较不同前缀时，会反复计算相同的子问题。

因此使用 DP 自底向上填表，避免重复计算。

### 2.2 DP 状态定义

设 `dp[i][j]` 表示 `text1` 的前 `i` 个字符 与 `text2` 的前 `j` 个字符 的 **最长公共子序列的长度**。

- `i` 的范围：`0 ~ m`（`m = len(text1)`）
- `j` 的范围：`0 ~ n`（`n = len(text2)`）
- `dp[0][*]` 和 `dp[*][0]` 初始化为 `0`，表示空字符串与任何字符串的 LCS 长度为 0。

### 2.3 状态转移方程

比较当前字符 `text1[i-1]` 和 `text2[j-1]`：

**情况一：字符相同**
$$
dp[i][j] = dp[i-1][j-1] + 1
$$
含义：当前字符可以加入前面的 LCS，长度 +1。

**情况二：字符不同**
$$
dp[i][j] = \max(dp[i-1][j],\ dp[i][j-1])
$$
含义：两个字符不能同时出现在 LCS 中，取**跳过 text1 当前字符**或**跳过 text2 当前字符**中的较大值。

---

## 三、代码详解

```python
from typing import List

class Solution:
    def longestCommonSubsequence(self, text1: str, text2: str) -> int:
        m, n = len(text1), len(text2)
        # 二维DP表，(m+1)行 (n+1)列，初始全0
        dp = [[0]*(n+1) for _ in range(m+1)]
        for i in range(1, m+1):
            c1 = text1[i-1]
            for j in range(1, n+1):
                c2 = text2[j-1]
                if c1 == c2:
                    dp[i][j] = dp[i-1][j-1] + 1
                else:
                    dp[i][j] = max(dp[i-1][j], dp[i][j-1])
        return dp[m][n]
```

### 代码逐段解读

| 代码 | 说明 |
|------|------|
| `m, n = len(text1), len(text2)` | 获取两个字符串的长度 |
| `dp = [[0]*(n+1) for _ in range(m+1)]` | 创建 (m+1) × (n+1) 的二维数组，所有元素初始化为 0。多出的一行一列用于处理空字符串的情况 |
| `for i in range(1, m+1):` | 从 1 开始遍历 text1 的每个字符（索引 i-1 对应实际字符） |
| `c1 = text1[i-1]` | 取出 text1 当前字符 |
| `for j in range(1, n+1):` | 遍历 text2 的每个字符 |
| `c2 = text2[j-1]` | 取出 text2 当前字符 |
| `if c1 == c2:` | 字符匹配 |
| `dp[i][j] = dp[i-1][j-1] + 1` | 左上角值 + 1 |
| `else:` | 字符不匹配 |
| `dp[i][j] = max(dp[i-1][j], dp[i][j-1])` | 取上方或左方的较大值 |
| `return dp[m][n]` | 最终答案在右下角 |

### 为什么 dp 数组要多一行一列？

多出的第 0 行和第 0 列代表**空字符串**。当 `i=0` 或 `j=0` 时，`dp[0][j] = dp[i][0] = 0`，这为递推提供了初始值，也让循环中的 `dp[i-1][j-1]`、`dp[i-1][j]`、`dp[i][j-1]` 在边界情况（`i=1` 或 `j=1`）不会越界。

## 四、实例演示

### 示例 1：`text1 = "abcde"`, `text2 = "ace"`

#### DP 表填表过程

|   |   | a | c | e |
|---|---|---|---|---|
|   | 0 | 0 | 0 | 0 |
| a | 0 | **1** | 1 | 1 |
| b | 0 | 1 | 1 | 1 |
| c | 0 | 1 | **2** | 2 |
| d | 0 | 1 | 2 | 2 |
| e | 0 | 1 | 2 | **3** |

**逐步推演：**

1. `i=1, j=1`：`text1[0]='a'` == `text2[0]='a'` ✅ → `dp[1][1] = dp[0][0] + 1 = 1`
2. `i=1, j=2`：`text1[0]='a'` ≠ `text2[1]='c'` ❌ → `dp[1][2] = max(dp[0][2], dp[1][1]) = max(0, 1) = 1`
3. `i=1, j=3`：`text1[0]='a'` ≠ `text2[2]='e'` ❌ → `dp[1][3] = max(dp[0][3], dp[1][2]) = max(0, 1) = 1`
4. `i=2, j=1`：`text1[1]='b'` ≠ `text2[0]='a'` ❌ → `dp[2][1] = max(dp[1][1], dp[2][0]) = max(1, 0) = 1`
5. `i=3, j=2`：`text1[2]='c'` == `text2[1]='c'` ✅ → `dp[3][2] = dp[2][1] + 1 = 1 + 1 = 2`
6. `i=5, j=3`：`text1[4]='e'` == `text2[2]='e'` ✅ → `dp[5][3] = dp[4][2] + 1 = 2 + 1 = 3`

✅ 最终结果：`dp[5][3] = 3`，LCS 为 `"ace"`。

#### 每个格子代表的含义

`dp[3][2] = 2` 的含义是：`"abc"`（text1 前 3 个字符）与 `"ac"`（text2 前 2 个字符）的 LCS 长度为 2，即 `"ac"`。

## 五、关键思想总结

### 5.1 填表方向

```
        text2 →
text1   ┌─────────┐
↓       │ ← ← ← ← │  每个格子依赖：
        │ ↑       │  • 左上角 (匹配时)
        │ ↖       │  • 上方 or 左方 (不匹配时)
        └─────────┘
```

**从左到右、从上到下**逐行填充，保证每个格子依赖的左、上、左上三个值都已经计算完毕。

### 5.2 「回溯」思想（可选扩展）

虽然本题只要求返回长度，但如果需要 **输出具体的 LCS 字符串**，可以从 `dp[m][n]` 反向回溯：

```
从右下角开始：
- 如果 text1[i-1] == text2[j-1]：该字符属于 LCS，记录它，向左上移动
- 否则：
  - 如果 dp[i-1][j] >= dp[i][j-1]：向上移动
  - 否则：向左移动
```

### 5.3 空间优化（可选）

当前解法使用 `O(m × n)` 空间。由于每个格子只依赖 **当前行** 和 **上一行**，可以将空间优化到 `O(min(m, n))`，使用两个一维数组滚动更新。

### 5.4 时间复杂度

- **时间复杂度**：`O(m × n)` —— 需要填充整个二维 DP 表
- **空间复杂度**：`O(m × n)` —— 二维 DP 表的大小

### 5.5 思维模型

> **把两个字符串想象成两排字符，DP 表记录了任意两个前缀之间的「最大匹配程度」。从小的前缀开始，逐步扩展到整个字符串，每一步只需做简单的局部决策。**

## 六、相关变体问题

| 问题 | 区别 |
|------|------|
| 最长公共子串 | 要求连续的公共子串，DP 转移方程不同（不连续时归零） |
| 最短公共超序列 | 找到包含两个字符串的最短序列 |
| 编辑距离（Levenshtein Distance） | 计算插入/删除/替换的最小操作次数 |
| 不同的子序列 | 计算一个字符串在另一个字符串中作为子序列出现的次数 |
